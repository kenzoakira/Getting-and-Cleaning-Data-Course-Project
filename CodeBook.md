**Transformation**

1.Combine train and test data and name them after the feature list.（X_train.txt,X_test.txt,features.txt）
2.Select only the columns, whose names only contain "mean()" and "std()".
3.Add three new columns - "Activity.Label" "Activity.Name"  "Subject.ID".(y_train.txt,subject_train.txt,y_test.txt,subject_test.txt,activity_labels.txt)
4.Create a data frame "strData" which contains all the information above.
5.Create a data frame "aggData", whose values are the aggregated  mean values group by the three new columns.
6.Export a txt file called "tidyData", which contains "aggData".
